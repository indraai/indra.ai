Methods

These methods are built into #VedaDeva to the Vedic Books.

Books

This command will return a listing of books for the Rig Veda from our custom Veda Library.

example#veda books

Book

This command will return the details and list of hymns in a specific book.

example#veda book [book number]

Hymn

Here we can view a specific hymn number fro the deva.world Vedic Library. The [hymn number] format is XX is the book number and XXX is the hymn number. The example below will view Book 01 Hymn 001.

example#veda hymn [hymn number]
example#veda hymn 01001
