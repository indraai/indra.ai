The Web Deva helps all the other Devas with their calling websites to get information. With the Web Deva you can give a url and the Agent will return the data response from your request. Be Careful this could be DANGEROUS!

Mehtods

cmd#web get *url*

This method will return the data from a specific url.

cmd#web json *url*

This method will return a json file from a specific url.

cmd#web rss *url*

This method will return a rss feed from a specific url.
